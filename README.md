# 📊 Supply Chain Performance Dashboard

An interactive **Supply Chain Analytics Dashboard** built using **Microsoft Excel** to analyze and monitor key business metrics such as revenue, profit, shipping costs, supplier performance, product sales, and defect rates.

---

## 🚀 Project Overview

This project helps businesses understand and improve supply chain operations through data-driven insights and interactive visualizations. The dashboard provides a clear overview of operational performance using charts, KPIs, slicers, and Pivot Tables.

---

## 🎯 Objectives

- Monitor overall supply chain performance
- Analyze revenue and profit trends
- Track shipping costs and lead time
- Evaluate supplier performance
- Identify defect rates and inspection results
- Compare transportation modes
- Analyze revenue by product category

---

## 🛠️ Tools & Technologies Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning & Analysis

---

## 📌 Key Performance Indicators (KPIs)

- Total Revenue
- Total Profit
- Shipping Cost
- Defect Rate
- Products Sold
- Lead Time

---

## 📈 Dashboard Features

### ✅ Revenue Analysis
- Location-wise revenue comparison
- Product category revenue analysis

### ✅ Supplier Performance
- Supplier-wise performance tracking
- Defect rate monitoring

### ✅ Shipping & Logistics
- Transportation mode analysis
- Shipping cost comparison

### ✅ Inspection Analysis
- Pass/Fail/Pending inspection results visualization

### ✅ Interactive Filters
- Product Type
- Supplier Name
- Location
- Transportation Mode
